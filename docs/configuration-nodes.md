# Configuration Nodes Reference

## Overview

This document summarizes available Behavior Tree nodes that are configurable via XML attributes and the common configuration keys used across the MAS-BT project. It includes examples for common node parameters and placeholder interpolation rules.

## Contents

- Node registry and naming conventions
- Property mapping: XML attributes to public properties
- Placeholder resolution: config and blackboard interpolation
- Examples: subscribing to topics, setting timeouts, precondition retries
